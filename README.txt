It's rating calculator for students created by V_Kn

RATING CALCULATOR - This application helps in calculating the ranking score at the University of Chernivtsi. Formula:
EAS * 0.9 + AP * 0.1 = Rating score
Ex. : (100+100+95+95)/4 * 0.9 + 10*0.1 = 88.75
The average score of exams, multiplied by 0.9 (success rate) and added up with additional points, multiplied by 0.1 (additional points coefficient).
The number of exams is calculated automatically.
